# Как писать в Markdown

* Нумерованный список вводится звёздочками
* Заголовки вводятся решётками: `#`, `##`, `###`
* Вставки кода вводятся `обратными кавычками`
* Листинги - тремя обратными кавычками подряд
```
#include <stdio.h>

int main() {
  printf("Hello\n");
  return 0;
}
```
